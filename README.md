Base Project for MEAN Stack WebApps 

#### How to use:

clone the repo using: 
```
git clone https://github.com/tiff-wang/mean_base
```

Make sure you have Node.Js and npm installed, if not please follow https://docs.npmjs.com/getting-started/installing-node

Install the node modules using
```
npm install 
```

The application is hosted on port 5000. To start the server using: 
```
node server.js
```  

You can also start the server using Nodemon. It will constantly watch your project for changes, and restart the node application server whenever one occurs. 
Follow https://www.npmjs.com/package/nodemon for your info!
